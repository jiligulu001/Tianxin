Apache Subversion and the Outreach Program for Women

Apache Subversion is taking part in the Outreach Program for Women 2013, a program initiated by the GNOME Foundation with the goal of increasing involvement of women in the development of Free and Open Source Software (FOSS). This page contains information about contributing to Apache Subversion as part of this program.

Elego Software Solutions, based in Berlin, Germany, has kindly offered to fund an internship position with Apache Subversion as part of the Outreach Program for Women 2013.

Please see our contributing page for general information about contributing to Apache Subversion. Our Community Guide is an essential read for new members of the community.

Project ideas ¶

  • Please see our project ideas page.

For more information about how to contribute, or to discuss your contribution with us, please contact dev@subversion.apache.org

Please note that the dev@ list is publicly archived, which means anyone can read any information sent there, and information sent there cannot subsequently be erased. If you have any private information to send as part of your application (such as a CV), you may send it directly to Subversion's OPW mentor, in addition to sending any other non-private information to the dev@ list.

The mentor for OPW 2013 for Subversion is Stefan Sperling <stsp@apache.org>, 'stsp' on Freenode IRC in the #svn-dev channel.
